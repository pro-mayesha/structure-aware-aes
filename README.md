# Structure-aware AES: Controlled Document-Structure Features for Essay Scoring

This repository supports a **controlled validity study** showing that discourse-derived document structure—especially **boundary geometry** and **rhetorical transitions**—adds predictive signal for holistic essay scoring **beyond** essay text, surface features, and component counts on **PERSUADE 2.0**.

## Important framing

- This is **not** a production-ready automated essay scoring system.
- This is **not** a state-of-the-art neural benchmark.
- This is an **oracle / upper-bound analysis** using **human discourse annotations** (character offsets and rhetorical labels). Structural features assume gold structure at scoring time.

**Primary experiment:** TF-IDF + Ridge, official train/test protocol (Steps 1–4).  
**Supplementary:** DeBERTa-v3-base robustness with frozen encoder when CUDA is unavailable (Step 5).

## Main results (official held-out test)

| Condition | QWK |
|-----------|-----|
| T2 (text + surface + counts) | 0.8361 |
| **T3_PLUS_POS** (full structure) | **0.8488** |

- **ΔQWK (T3_PLUS_POS − T2):** +0.0128; bootstrap 95% CI [+0.0092, +0.0162]; P(Δ > 0) = 1.000  
- **Prompt consistency:** T3_PLUS_POS beats T2 on **15/15** prompts  
- **Falsification:** Genuine structure beats **shuffled labels**, **random boundaries**, and **capacity-matched random features** on test

## Quick start

### 1. Environment

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python scripts/smoke_check.py    # no dataset required
```

### 2. Data (obtain separately)

Download PERSUADE 2.0 from [REALISE](https://github.com/scenario-nlp/REALISE) and place:

```text
data/persuade20/raw/persuade_corpus_2.0_train.csv
data/persuade20/raw/persuade_corpus_2.0_test.csv
```

### 3. Reproduce

```bash
make audit                 # Step 1 — requires raw data
make reproduce-all         # Steps 1–4 (main paper pipeline)
make reproduce-transformer # Step 5 optional
```

Or run scripts directly — see [`REPRODUCIBILITY.md`](REPRODUCIBILITY.md).

## Data policy

**Raw PERSUADE 2.0 files are not redistributed.** Obtain the corpus under its own license. `.gitignore` blocks accidental commits of raw CSVs, processed essays, embedding caches, and model checkpoints.
